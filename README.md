# alt-f4-disable
a autohotkey script to disable alt-f4 (with a funny textbox)

# installing

## manually

go to the releases

move nof4.exe to the startup folder

# uninstalling

go to `C:\Users\%username%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup` on explorer

delete nof4.exe
